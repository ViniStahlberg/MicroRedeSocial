## Sobre o Projeto

Aplicativo Android de uma micro rede social desenvolvido para a disciplina **ARQDMO2 - Dispositivos Móveis 2** no **IFSP - Campus Araraquara**.

## Funcionalidades

✅ Login e cadastro com Firebase Authentication  
✅ Criar postagens com texto, imagem da galeria e localização via GPS  
✅ Feed com paginação (5 posts por vez)  
✅ Buscar postagens por cidade  
✅ Editar perfil (nome, foto e senha)  

## 🛠️ Tecnologias

- Kotlin
- Android API 33
- Firebase Authentication
- Firebase Firestore
- Google Play Services (Localização)
- Material Design
- ViewBinding

## Como Executar
1. Clone o repositório
2. Abra no Android Studio
3. Adicione o arquivo `google-services.json` na pasta `app/`
4. Execute o app
